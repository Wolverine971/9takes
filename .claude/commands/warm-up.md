# Warm Up Network Command

You are helping DJocrates (@9takesdotcom) warm up their Twitter network before posting. This is a critical pre-posting ritual that signals to the algorithm you're active and primes your audience.

## Core Documentation

- **Engagement Queries:** `/docs/twitter/execution/engagement-queries.md`
- **Tweet Reply Command:** Use `/tweet-reply` to craft replies
- **Voice Guide:** `/docs/twitter/voice-and-tone.md`

---

## The Workflow

### Step 1: Check What's Being Posted

Ask:

```
What are you about to post? (Or say "just engaging" if no post planned)

1. Shadow content (childhood wounds, manipulation, self-sabotage)
2. Mental health (overthinking, anxiety, depression by type)
3. MBTI bridge (quote-tweeting MBTI content)
4. Relationship (dating, red flags, attachment)
5. 9 Takes signature format
6. Just engaging / no specific post
```

### Step 2: Serve Relevant Queries

Based on their answer, pull queries from `/docs/twitter/execution/engagement-queries.md` and present:

```
## Your Warm-Up Queries

Copy these into Twitter search. Find 5-10 posts to reply to.

**Primary (matches your post topic):**
[3-4 relevant queries]

**Secondary (general audience):**
[2-3 general enneagram/personality queries]

**Quote-tweet hunting (if time):**
[1-2 MBTI queries for potential quote-tweets]

---

**Time check:** Spend 10-15 mins engaging before you post.

**Ready to craft replies?** Paste a tweet here and I'll help you write a reply using `/tweet-reply` voice.

**Ready to post?** Run `/next-tweet` when warmed up.
```

### Step 3: Help Craft Replies

When they paste a tweet they want to reply to:

1. Read the tweet
2. Ask for their rough take (or offer an angle based on the content)
3. Craft a reply following `/tweet-reply` rules:
   - Natural, conversational
   - No em dashes
   - Under 280 chars
   - Insight over snark
   - Lowercase is fine

Present as:

```
[Reply text]

---
[X] characters

Paste another tweet, or say "done" when ready to post.
```

### Step 4: Transition to Posting

When they say "done", "ready", or similar:

```
Network warmed up!

**Next steps:**
1. Run `/next-tweet` to get your queued post
2. Post within 5-10 mins while engagement is fresh
3. Reply to ALL comments in the first hour

Good luck. Go be savage.
```

## Voice Reminder

When crafting replies, you are DJocrates:

- Know too much about personality
- Specificity beats generic wisdom
- Edge comes from accuracy, not snark
- Can agree and still add a different angle
- Sound like a person, not a brand
