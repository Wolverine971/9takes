# SEO Gap Analysis: Joe Rogan

## Search Frame

- Primary query: Joe Rogan personality type
- Secondary queries: Joe Rogan enneagram, why Joe Rogan interviews work, Joe Rogan personality analysis
- Search intent: informational with light comparative intent
- Reader job: understand his personality pattern quickly and decide whether the analysis feels evidence-based

## Current Asset Audit

| Area            | Status  | Notes                                                       |
| --------------- | ------- | ----------------------------------------------------------- |
| title           | mixed   | works editorially, could sharpen query match                |
| meta_title      | weak    | not fully exploiting search phrasing                        |
| description     | weak    | too short to set payoff                                     |
| H2/H3 structure | mixed   | vivid, but several headers are query-light                  |
| FAQ coverage    | missing | no snippet support                                          |
| internal links  | mixed   | could connect more directly to related personality profiles |

## External Pattern Snapshot

| Pattern                         | Seen often? | Why it matters                          |
| ------------------------------- | ----------- | --------------------------------------- |
| direct "what type is X?" titles | yes         | searchers want immediate answer framing |
| FAQ-style sections              | yes         | helps snippet capture and scannability  |

## Gap Map

### Must-have gaps

- add FAQ block
- sharpen query-bearing H2 language

### Differentiation gaps

- lean harder into motive plus fear, not just public persona

### Internal link gaps

- link to adjacent media/podcaster profiles and enneagram authority pages

## Recommended Fixes

- New title: Joe Rogan Personality Type: What His Enneagram Pattern Reveals
- New meta title: Joe Rogan Enneagram Type and Personality Analysis
- New description: A direct, evidence-based read on Joe Rogan's personality pattern, communication style, blind spots, and why his interviews work.
- Suggested FAQ questions:
  - What is Joe Rogan's personality type?
  - Why does Joe Rogan interview people this way?
- Suggested H2/H3 replacements:
  - "The Conversation That Never Ends" -> "Why Joe Rogan's Interview Style Feels So Intense"
- Suggested internal links:
  - related podcaster and contrarian-thinker analyses

## Priority

- Impact: high
- Effort: medium
- Best next move: structural SEO pass before any deep rewrite
