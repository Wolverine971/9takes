## 2. The reusable skeleton (lookup-table teaching engine)

This is a different engine from the "Weird AI" post. That one taught _judgment_ (9 beats building to a checklist). This one is a **taxonomy / lookup table** — its value is the matrix you can screenshot and reuse forever:

1. **Cover names the artifact, not the topic** — "The [X] Cheat Sheet." Promises a _reference_, not a lecture. The implicit deal: save this and you never have to think about it again.
2. **Problem hook = sameness** — "Most [outputs] look the same. Not because of [the obvious scapegoat]. Because everyone [makes the same single choice]." Locates the reader in the mediocre majority, then offers the exit.
3. **One-line thesis as an equation** — "[Technique] = EMOTION." The whole deck is this claim, repeated as data.
4. **The entry pattern, repeated N times** — every taxonomy item is a _pair_ of slides:
   - **Teach card:** `#N: NAME` → _Use when_ (the trigger) → _Best for_ (chip word-bank) → _Creates_ (chip word-bank of emotions).
   - **Example card:** full-bleed photo + a single overlay badge `[NAME] = [Emotion]`. The badge is the atomic, shareable unit.
5. **"Quick rule" interstitials** — inserted every ~2 entries. Compress the choice to a binary `Want X? → Do A. Want Y? → Do B.` They pace the deck and reframe the system as a decision, not a reference.
6. **"Save this formula" recap** — the entire taxonomy as one list on one slide. This is the screenshot the whole deck exists to produce.
7. **Comment-bait CTA** — a question + a keyword trade for "more cheat sheets."

**Teach/show rhythm is the core mechanic.** Concept slide → proof slide → concept → proof. The reader never has to _imagine_ the result; the next slide shows it with a one-word label. Half the slides are doing the teaching; half are doing the convincing.

## 3. Visual / production DNA

- **Consistent chrome** top + bottom on every slide (brand, `AI Camera Angles` tag, URL). Same discipline as the Weird AI post.
- **Cool, cinematic palette** — blue/black/near-monochrome photography with a single red CTA gradient. (9takes inverts this: warm amber Streetlamp, dark text on amber.)
- **Pill / chip word-banks** for _Best for_ and _Creates_ — visually distinct from headline text, scannable, screenshot-able.
- **One overlay badge per example slide** (`X = Emotion`) — tiny, lower-third, the recurring brand tic that ties teach card to proof card.
- **Heavy condensed sans** for `#N: NAME` and headlines; **italic serif** reserved for the cover subhead ("Learn which angle creates which emotion") and "Steal these angles" moments.
- Every example photo is itself an instance of the rule — the medium proves the message.
- 1:1 square, text top-weighted to survive the feed crop.

## 4. The transferable insight

The engine is: **take something practitioners do by feel and turn it into a one-to-one map (technique → emotion).** People save lookup tables because they outsource a recurring decision. The "cheat sheet" framing is the value promise.

For 9takes the port is exact, because the Enneagram _is_ a technique→emotion lookup table for human behavior, and the source's own thesis is already the 9takes tagline:

> "The same subject can feel powerful, vulnerable, or intimate simply by changing perspective."
> = **"One situation, nine ways to see it."**

iklipse keeps the subject and moves the **camera**. 9takes keeps the moment and moves the **mind**. Same revelation, pointed at psychology instead of optics. Two ways to build it:

- **Direct analog — "The Enneagram Cheat Sheet":** each type is a lens. Teach card (`TYPE 8 — THE CHALLENGER` / _Leans in when_ / _Drawn to_ / _Creates_) + example card (Greek-statue portrait + `Type 8 = Control` badge). Evergreen series engine — nine clean, reusable entries.
- **Contrast play — "9 Reactions, 1 Moment":** freeze one scenario ("You got passed over for the promotion"), then nine identical-frame slides where only the number + reaction change. This _is_ the give-first reveal in carousel form and demos the product mechanic directly. Stronger differentiator; weaker as a repeatable template.
