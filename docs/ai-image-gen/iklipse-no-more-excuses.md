## Beat skeleton

Hook (blunt "no more excuses") → **3× objection cards, each = a quoted excuse + a one-line rebuttal** → "stop making excuses" comment CTA.

The engine is **objection-handling**: each slide voices the reader's own excuse in quotation marks ("I don't have the budget/team/time"), then dismantles it in a single line. The quoted excuses make it feel like the deck is reading the viewer's mind — which is what makes objection-handling persuasive. This is the **sales-page "but what about…" structure** compressed into a carousel.

## Highest-value extractables

- **Quoted-excuse + rebuttal card** — voice the reader's exact objection ("I don't have X"), then demolish it. Each slide is one excuse. Highly reusable persuasion frame.
- **The "big three" objections** (budget / team / time) are universal — the same trio blocks most action in any domain.
- **"Stop making excuses" accusatory CTA** — mild confrontation that invites a defiant/agreeing comment.
