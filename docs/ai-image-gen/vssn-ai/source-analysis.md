## 4. The reusable caption skeleton

Post B is a clean, liftable template. The beats:

1. **Triple-negation hook** — "No model was hired. No camera was used. No one left the house." (curiosity through subtraction)
2. **Early comment-CTA** — "Comment PROMPTS and steal every single one 👇" (placed _before_ the proof, while attention is highest)
3. **Triple-affirmation** — "Real outfits. Real locations. Real energy." (rebuts the disbelief the hook created)
4. **The reframe / doctrine line** — "Zero cameras. Just words in a text box."
5. **Imperative trio** — "Save it. Use it. Generate it yourself." (explicit save command — the saves lever)
6. **Tool credit** — "Built with GPT Image 2.0" (credibility + SEO)
7. **Niche hashtag stack** — 5 tags, all on-topic, no spray.

Rule of three is doing the heavy lifting (negation / affirmation / imperative). The CTA sits high, not at the end.

## 5. Visual / production DNA

- **Magazine-cover format** for prompt packs ("PROMPTS — NEW WAVE — VOL.01 ISSUE 01", "STEAL PROMPTS"). Masthead typography, barcode, issue number — packaging the asset as a collectible. This is a _cover_ idea worth stealing (high save value).
- **Contrarian hook covers** as the dominant grid pattern: "they think its real," "Your AI Footage Looks Fake," "VOGUE SPENT MILLIONS TO SHOOT THIS.", "THE BAR FOR AI PORTRAITS JUST MOVED.", "I BUILT THAILAND FROM PROMPTS." Each is a status / disbelief / FOMO trigger.
- Palette: cinematic, desaturated, editorial. Off-brand for 9takes (we stay amber Streetlamp) — take the _format_, not the color.
