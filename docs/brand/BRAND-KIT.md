## Typography

### Primary Font: Noticia Text (Serif)

The site uses **Noticia Text** for all text - body, headings, and even code/mono contexts.

**Google Fonts Alternative**: If Noticia Text isn't available in Canva, use **Lora** or **Playfair Display** (similar serifs).

**Font Files Location**: `static/fonts/`
- NoticiaText-Regular.woff2 / .woff / .ttf
- NoticiaText-Bold.woff2 / .woff / .ttf
- NoticiaText-Italic.woff2 / .woff / .ttf
- NoticiaText-BoldItalic.woff2 / .woff / .ttf

### Secondary Font: JetBrains Mono (Code Only)

Used only in admin code editors and markdown editing. Not needed for marketing materials.

## Brand Voice

### Taglines

**Primary**: "See the emotions behind every take"

**Secondary**: "One situation, 9 ways to see it"

**Positioning**: "Decode social dynamics, personality-max your EQ, turn conflict into understanding"

### Voice Attributes

1. **Tactically Direct** - No fluff, actionable information
2. **Socially Savvy** - Connect insights to real-world wins
3. **Respectfully Provocative** - Challenge comfort zones
4. **Pattern-Recognition Focused** - Show emotional logic
5. **Results-Driven** - Focus on outcomes

### Writing Rhythm

**Hook** → **Insight** → **Action Step**

### Key Verbs

Use these action words in copy:
- Decode
- Navigate
- Map
- Read
- Unlock
- Resolve

### Tone Guidelines

| Do | Don't |
|----|-------|
| Be direct and clear | Use corporate jargon |
| Show patterns and logic | Make vague claims |
| Challenge with respect | Be preachy or judgmental |
| Provide actionable steps | Give fluffy advice |
| Use conversational language | Sound academic |

## Canva Setup Instructions

### 1. Brand Colors

Add these as your brand palette in Canva:
```
Primary: #7c3aed
Secondary: #3b82f6
Accent: #06b6d4
Background Dark: #0a0a0f
Background Card: #1a1a24
Text Light: #f8fafc
Text Secondary: #94a3b8
Gold: #d4af37
```

### 2. Fonts

**In Canva Brand Kit:**
- Headline: Upload Noticia Text Bold (or use Lora Bold)
- Body: Upload Noticia Text Regular (or use Lora Regular)

### 3. Upload Assets

Upload these files to Canva Brand Kit:

**Logos:**
- `static/brand/aero.png` (Main logo - blue tint)
- `static/brand/darkRubix.png` (Icon/mark)
- `static/brand/nimbus.png` (Light variant)

**Graphics:**
- `static/enneagram.svg` (Enneagram symbol)
- Other logo variants as needed (aria, fresco, oceanic, polar)

### 4. Templates

Create templates using:
- Dark background (#0a0a0f)
- Purple gradient overlays (#7c3aed to #5b21b6)
- Cyan accent highlights (#06b6d4)
- White/light text (#f8fafc)

---

## Gradient Recipes

### Primary Gradient
```css
background: linear-gradient(135deg, #7c3aed 0%, #5b21b6 100%);
```

### Glow Effect
```css
box-shadow: 0 0 20px rgba(124, 58, 237, 0.4);
```

### Card Background
```css
background: linear-gradient(180deg, #1a1a24 0%, #12121a 100%);
border: 1px solid rgba(124, 58, 237, 0.2);
```

### Text Glow
```css
text-shadow: 0 0 10px rgba(124, 58, 237, 0.5);
```
