## Content Approach

### Dual-Title System (Celebrity Content)

| Field        | Purpose        | Style                    |
| ------------ | -------------- | ------------------------ |
| `title`      | Page headline  | Evergreen, authoritative |
| `meta_title` | Search results | Clickbait, CTR-optimized |

### Writing Rhythm

```
Hook · Insight · Action step
```

### Key Verbs

Decode, navigate, map, read, unlock, resolve

### Never Use

- Academic/theoretical tone
- Hedging language ("tend to", "might")
- Generic descriptions
- Unsubstantiated claims
