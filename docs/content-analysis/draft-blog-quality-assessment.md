## Quality Rubric (Condensed)

- Title: problem/question hook, <= 60 chars, includes person name; meta_title optimized for clicks
- Meta description: 150-160 chars, Problem + Promise + Curiosity hook
- QuickAnswer component near top with direct 2-3 sentence answer
- Opening: firstLetter hook, restate core question, promise answer, zero preamble
- Body: H2 "What is [Person]'s Personality Type?" with H3 answer; type-specific depth with evidence and quotes
- Remove hedging language (tend to, might, can be, perhaps) and avoid generic claims
- Add FAQ section (4-5 real search questions) before disclaimer
- 3-5 internal links (type page + related people + mental health/related content)
- Disclaimer at end; preserve Enneagram type and voice
