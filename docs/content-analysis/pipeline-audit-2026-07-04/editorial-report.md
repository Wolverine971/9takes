# Editorial-Judgment Audit — Lanes 1-4 (Steve Martin, Jeff Goldblum, Tobey Maguire, Zac Efron, Rosé, Mira Murati)

Read-only audit of current editorial output quality. The 2026-07-01 grade-inflation fixes are assumed working (they are — the grader now honestly caps and names real issues). The question here is what the honest grades reveal about the ceiling.

**Meta-finding up front:** The grader is now trustworthy, and its honesty exposes a _structural ceiling_. The pipeline reliably manufactures an 8.2-8.5 artifact — same furniture inventory, same empathy-turn grammar, same monocausal childhood-wound risk, same untagged-diagnosis-cluster. Mira Murati (8.8) is the only one to break out, and she broke out because the _world_ handed her news-grade verifiable material (sworn court testimony, a viral clip, a live drama), not because the pipeline built her differently. Threshold-chasing is no longer the disease; the template ceiling is.

### Proposed testable definition of "one-of-one" (Deliverable #3)

The stated test is a necessary _floor_, not the bar. A draft is one-of-one only if it passes **all three**:

1. **Detail floor (necessary, already passing):** ≥5 proper-noun-independent subject-specific details identify the subject. All 6 pass — keep as a minimum gate, not the standard.
2. **Argument non-transfer (the discriminating check, scriptable):** With proper nouns stripped, the diagnosis + counter-typing + closing beat must not reuse a same-type sibling's argument moves. Deterministic proxy: similarity-scan those three passages against the last N same-type drafts; hard-fail on any shared counter-typing _tiebreaker sentence_, shared arrow explanation, or shared closing shape. Concrete rule: **no two same-type drafts may share a counter-typing tiebreaker.** (Zac + Mira fail this today.)
3. **One bespoke section form (structural):** at least one body section whose _structure_ — not just its nouns — is generated by the subject's world and would collapse if transplanted. Goldblum's The Fly section and Mira's two-years-apart quote rhyme qualify. Zac and Tobey have none.

Under this definition today: Goldblum PASS, Mira PASS, Rosé borderline-PASS, Steve borderline, **Tobey FAIL, Zac FAIL**.

### The pattern

The revision loop reliably tags **the one quote the grader names** (usually the cold-open/epigraph) and stops. Diagnosis-cluster and empathy-payload quotes stay untagged. Evidence 8-vs-9 is effectively "did the single most-named quote get tagged," not "are all load-bearing quotes traceable." This is textbook "cleared by tactical source fix."

Also: the `citations` frontmatter **launders**. Every draft's citations are mostly Wikipedia URLs, not the Guardian/Rolling Stone/NPR/WSJ outlets actually quoted inline. The frontmatter manufactures an appearance of 5-10 sources that doesn't map to the inline quotes.

### Proposed source-quality standard (Deliverable #6)

Define **load-bearing quote** = any quote in one of five slots: (a) epigraph, (b) cold open, (c) diagnosis/type-call section, (d) empathy turn, (e) closing beat.

- **B+ eligibility:** ≥3 inline-dated tags (outlet + year) in the body; **zero untagged quotes in the epigraph or cold open**; no sensitive claim (crime, health, addiction, sexuality, contested allegation) about a living person without a named source.
- **A eligibility:** **every** load-bearing quote (all 5 slots) carries inline outlet + year; ≥5 inline-dated tags total; every sensitive claim carries a _top-tier named source inline_ — no "a rival's memo," no frontmatter-only.
- **A+ eligibility:** all of A, plus ≥1 quote traceable to a _primary_ source (raw transcript, memoir, court testimony, the person's own writing), and a machine-checkable evidence packet mapping each load-bearing quote to a URL.

**Deterministic pre-grade check:** a "load-bearing quote audit" enumerating each of the 5 slots, its attribution status (outlet Y/N, year Y/N, URL Y/N, source class), and hard-blocking A-grade if any slot has an untagged quote. Converts Evidence from a vibe to a checklist and kills the "tag only the named quote" behavior. Add a citations-vs-inline reconciliation: flag when `citations` are mostly encyclopedia backfill and omit the outlets quoted inline.

Subtract Glenn and you still have a Five") and de-monocaused it in a single paragraph.

### Wings/arrows: mostly Rabbit-Hole filler; counter-typing is the one element earning its place

The four-part Rabbit Hole (Wing / Subtype / Arrows / Counterarguments) is identical across all six. The **arrow** sections mostly restate the body ("under stress the Five moves to 7 — see the panic years"); they rarely reveal something new. The **subtype** sections are the weakest and most interchangeable ("self-preservation dominant: the fortress life"). The **wing** calls are asserted with thin evidence. Only the **counterarguments** subsection does real analytical work.

### Recommendation: pre-write type-challenge memo — YES

Required contents:

1. Strongest case FOR the assigned type — 3 pieces of behavioral evidence.
2. Strongest case AGAINST — the best alternate type, with its own evidence.
3. A named **tiebreaker** — the specific behavior the assigned type explains and the alternate cannot.
4. One **"behavior this type does not explain well"** (the command's own suggested intervention, line 171).
5. An explicit **confidence level** (high / medium / low).

**What should be able to change the type before drafting:** if the tiebreaker behavior is absent or ambiguous, OR the "doesn't explain" list holds 2+ major behaviors, the memo escalates to a human type decision instead of proceeding on the queue's preassignment.

Plus a **grader monocausal cap:** if a draft routes 3+ distinct major behaviors to a single childhood wound with no secondary thread, cap Enneagram at 8. Steve's revision shows the fix is one paragraph.

## Minimum viable deterministic checks (Deliverable #4)

1. **Contrast-pair / negative-parallelism counter in blog-lint** — regex for `not X[,.] it (was|is) Y`, `wasn't X; it was Y`, `not X but Y`, `less X than Y`. Replaces the self-reported, chronically-stale Formula Fingerprint Ledger. Also catches the empathy-turn grammar.
2. **Load-bearing quote audit** (Lane 2) — enumerate the 5 slots, check inline outlet+year+URL, hard-block A-grade on any untagged slot.
3. **Same-type argument-similarity scan** (Lane 1) — diagnosis + counter-typing + close vs. last N same-type drafts; hard-fail on a shared tiebreaker. (Catches Zac↔Mira.)
4. **Citations-vs-inline reconciliation** — flag when `citations` are mostly Wikipedia and omit the outlets quoted inline.
5. **Bespoke-section presence** — at least one H2 outside the house slot-set {diagnosis, childhood-thread, walked-away/arc, empathy-turn, thaw/close}.

## Bottom line for the orchestrator

The grader is honest; the honesty reveals a template ceiling around 8.2-8.5. The four biggest levers, in order: (1) make the contrast-pair count deterministic so Writing stops being perma-capped and the editor actually finishes its job; (2) a load-bearing quote audit so Evidence hits 9 on _all_ load-bearing quotes, not just the one the grader names; (3) a bespoke-section requirement + same-type argument scan so Originality escapes the type-template (kills the Zac↔Mira Mad-Libs); (4) a pre-write type-challenge memo with a confidence level so the queue type is a hypothesis, not a verdict. Subject selection is the fifth, invisible lever — Mira proves that news-grade verifiable material buys a grade the template alone cannot.
