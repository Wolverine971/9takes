## Key Optimization Protocol (Quick Reference)

For any content optimization, apply these 9 steps:

1. **Title Rewrite** - Problem-focused, max 60 chars
2. **Meta Description** - Problem + Promise + Hook, 150-160 chars
3. **Add Quick Answer Box** - Direct answer in first 50 words
4. **Rewrite Opening** - Name problem, validate, promise solution
5. **Remove Hedging** - "tend to" → "typically", "might" → "often"
6. **Add FAQ Section** - Real search queries from GSC
7. **Update Internal Links** - 2-3 links to/from related content
8. **Update Frontmatter** - lastmod to today
9. **Verify Quality** - All checks pass

See `celebrity-page-optimization-instructions.md` for full details.
