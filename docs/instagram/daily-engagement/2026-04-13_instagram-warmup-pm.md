**Internal Pattern Read:**
Each Enneagram type has a specific override story — a narrative that makes ignoring the body signal feel rational, even virtuous:

- Type 1: "I can't stop, there's still work to do. Rest is when it's done." The signal gets filed under "later."
- Type 2: Someone else's need lands louder. "They need me more than I need rest." The body's signal becomes a guilt signal.
- Type 3: Busyness IS the identity. Slowing down feels like stopping. The signal never even fully registers before the next task appears.
- Type 4: "I'm feeling things intensely — that's just how I am." The emotional noise drowns the body signal.
- Type 5: Intellectual override — the body is managed via rational analysis. "I'll address this later, logically."
- Type 6: The signal is scanned for threat. "Is this serious? Should I worry? Maybe I'm overreacting." The anxiety about the signal replaces the signal itself.
- Type 7: "Something better is coming — pivot." The body's message gets redirected into anticipation of the next thing.
- Type 8: "Intensity is normal. Push harder." The signal is treated as weakness to override rather than information to integrate.
- Type 9: "It's probably fine. I'll deal with it when I have to." The signal gets merged into ambient calm until it can't be ignored.

The Enneagram doesn't just show the override — it shows WHY the override feels like the right choice, and what that reveals about the type's core wiring.

- The Enneagram thing here is that each type doesn't just override the signal — they have a specific story that makes the override feel justified in the moment. Type 3 keeps running because stopping feels like stopping. Type 9 tunes it out because the signal feels like unnecessary noise in an otherwise fine situation. Same signal, completely different logic for why it can wait.
