# Instagram Carousel: The Three Pillars

_Created: 2026-03-12_
_Account: @9takesdotcom_
_Format: 10-slide carousel_
_Dimensions: 1080x1350px (4:5 portrait)_
_Visual direction: Dark background (#0a0a12), bold white/gold text, Solo Leveling aesthetic_

---

## Slide 1 — Cover

```
EVERY PERSON
WANTS 3 THINGS

(and they map to the
3 types of intelligence)
```

**Design notes:** Dark background. "3 THINGS" in large bold gold/accent text. Subtle glow behind the text. Clean, minimal — intrigue-first. Small 9takes logo bottom-right.

---

## Slide 2 — The List

```
1. SEE PROBLEMS COMING
2. KNOW WHAT TO DO
3. FEEL UNDERSTOOD

These aren't personality quirks.
They're hardwired.
```

**Design notes:** Numbered list in bold white. Each number could have a subtle icon (eye, fist, heart). Bottom text in muted gray. Left-aligned for readability.

---

## Slide 3 — The Framework

```
THE ENNEAGRAM MAPS
3 INTELLIGENCE CENTERS

Head  →  Foresight
Gut   →  Agency
Heart →  Belonging
```

**Design notes:** Three rows with arrow connectors. Could use a simple triangle diagram or three glowing nodes. Each center gets its own accent color (blue for Head, red for Gut, purple for Heart).
