## Reminders / context to carry forward

### Realistic reply expectations

- **Across 11 cold sends (excluding Rogan): 0–1 actual replies statistically.** Don't be discouraged by silence.
- If you want more replies, the lever is **warm intros, not better copy.** ([Noah Kagan got 0% from 50 cold pitches](https://noahkagan.com/how-to-connect-with-top-influencers/))
- Mutual connections in this network: Williamson/Peterson/Huberman/Bartlett all interview each other constantly. If ONE replies, ask for an intro to the others.

### Subject line note on the T1 gatekeeper inboxes

For Chris (`jonathan@`), Steven (`press@`), Peterson (`media@`), Shawn (`shawnryan@`):

- These hit gatekeepers, not principals.
- Subject line has to work for TWO readers: gatekeeper (does this earn a forward?) and principal (if forwarded, does it earn a click?).
- The `[Name] — a [wing] read of you` format does both — personalization shows the gatekeeper this isn't spam; specificity makes the principal click.

### Follow-up cadence (corrected from original)

- First send → wait **5–7 days** → one follow-up with the adjacent URLs as new value-add
- If still no reply after follow-up → wait to day 14 → optional second follow-up (only if you have new context to offer)
- After two no-replies → stop. Move on.

### Reply handling

- "Why are you writing about all of us?" → "The long-form interview format is doing real cultural work right now. I think the people running these shows deserve serious analysis, not just clips. You're 12 of those people."
- Substantive feedback ("the X part is off") → fix the piece. Reply thanking them. Don't ask for anything else.
- "Cool, thanks" or short positive → don't push. Leave the door open.
- Negative / hostile → don't engage publicly. Reply once with "fair enough, thanks for reading." Drop.
