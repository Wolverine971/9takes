---

## Search Queries to Run

### Direct Enneagram + Celebrity Searches

Run these searches on Twitter/X and look for people who write substantive analyses (not just "omg she's such a 3"):

```
"enneagram type" "taylor swift"
"enneagram type" "elon musk"
"enneagram type" celebrity
"enneagram" "personality type" famous
"enneagram" analysis thread
"enneagram type 1" OR "type 2" OR "type 3" OR "type 4" celebrity
she's a type 3 OR he's a type 8 OR they're a type 5
```

## What to Look For

**Green flags** (recruit this person):

- Writes multi-tweet threads analyzing a person's type
- Uses specific evidence (quotes, incidents) not just stereotypes
- Shows nuanced understanding (wings, subtypes, stress lines)
- Engages in typing debates with curiosity, not dogma
- Has their own blog, Substack, or newsletter about personality

**Red flags** (skip):

- Only posts memes about types
- Uses Enneagram as astrology ("classic 4 move lol")
- Gets hostile in typing disagreements
- Only tweets about their own type, not others'
- Account is mostly retweets with no original analysis

## Process After Finding Prospects

1. **Engage first** — Like or reply to 2–3 of their tweets before DMing. Don't cold-DM someone you've never interacted with.
2. **Personalize every DM** — Reference a specific tweet or thread. Never send the template verbatim.
3. **Follow up once** — If no reply in 5–7 days, one follow-up is fine. After that, move on.
4. **Track in a spreadsheet** — Name, handle, what they wrote that caught your eye, date contacted, response status.
