# The Mirror Moment

> **Status:** Canonical spec. Distilled 2026-07-01 from its first use — the Chappell Roan story carousel (`docs/ai-image-gen/moodboards/chappell-roan-moodboard.md` Shot 7 + `docs/ai-image-gen/moodboards/chappell-roan/carousel.md` slide 10), which referenced this doc before it existed.
> **Kin:** the give-first reveal and question-first funnel in [`the-chorus-vision.md`](./the-chorus-vision.md). The Mirror Moment is the social-media doorway version of the same move: the reader's own take is the product.

## The one-sentence mechanic

Every 9takes story carousel ends by **turning the lens off the subject and onto the reader** — the subject's pattern becomes a question about _yours_, and the only place to answer it is a 9takes question page.

The deck spends eight slides putting you behind someone else's eyes; the last slide asks what _you_ see. That's the product thesis ("see the emotions behind every take") executed as a slide order.

## The rules

### 1. Placement + silence

- The Mirror Moment is the **last slide**, after the type reveal.
- It carries **no CTA**. The silence after the question is the mechanic — the reader answers in their head before they're asked to do anything. The CTA lives in the caption and the bio link.

### 2. The image — an empty frame

- **No person, no face, no human reflection anywhere.** The subject's space with the subject gone — the vacated vanity stool, the empty desk chair, the notecards with no hand.
- **Rhyme it against one earlier slide.** Same room the reader just lived through, now vacated and turned slightly toward the viewer: _this seat is for you._
- **Generous negative space** for large centered overlay text. Mood: quiet and invitational, not sad or spooky.
- Shared moodboard signature still applies (amber sliver, film grain, one dominant light source) — but **no fact caption**; the question is the only text.

### 3. The text — bridge line + question

- **Bridge line** (small, on top): universal, no subject name. It generalizes the subject's pattern so the question needs no context. Example: _"We all build something to survive. The hard part is remembering how to take it off."_
- **The mirror question** (large, centered): traces back to the deck's through-line, and is **near-verbatim to the title of the 9takes question page** it drives to. Example: _"What's the armor that once protected you — and now keeps people out?"_

### 4. The 9takes-only test (all four, or rewrite)

A mirror question earns the slot only if it is:

1. **Non-binary** — no yes/no, no pick-a-number.
2. **Demands a narrative** — a real answer is a story, not a word.
3. **About the reader**, not the subject.
4. **Too exposed to answer in public comments** — which is exactly why it drives to 9takes, where answers are anonymous and give-first.

### 5. The landing — question page, not blog

- **Bio link → the 9takes question page titled with the exact mirror question.** Not the analysis blog. The question comes first; the analysis is the optional follow-up read after they write their take.
- **Caption bridge** names the tension without answering it: _"You just answered that in your head, didn't you? Write it down where it counts. 👇"_

## Worked example (Chappell Roan, Type 8)

| Layer           | Instance                                                                                                       |
| --------------- | -------------------------------------------------------------------------------------------------------------- |
| Through-line    | The armor has a dead man's name on it — strong everywhere except love.                                         |
| Rhymed frame    | The backstage vanity where she became the persona (slide 6) — now emptied, stool turned toward you (slide 10). |
| Bridge line     | "We all build something to survive. The hard part is remembering how to take it off."                          |
| Mirror question | "What's the armor that once protected you — and now keeps people out?"                                         |
| Landing         | Question page titled with that exact question; analysis blog linked from there.                                |
