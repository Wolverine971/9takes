- **Built-in-contrast questions are the highest-leverage shape**. #1 ("shut down vs lash out") and #2 ("hate vs thrive under authority") and #6 ("distracted vs not") all have "while others" framing. These let us answer with two-or-three-type structure without it feeling forced. Type-comparison is exactly what Enneagram is built for — these questions are pre-built for our angle. Prioritize this question shape in future warmups.
- **Type 7's biggest gap was always "starting-not-finishing"**, even though we've covered novelty/busyness/stillness-avoidance separately. This is the gateway behavior most people recognize first. Should have been queued earlier.
- **Phase A standard searches stay saturated**. Skipped people-pleaser/codependency/savior/ghosting/perfectionism searches because the question log shows those clusters are deeply covered. The session-by-session yield from Phase A has dropped sharply since 04-23.
- **Type 5 needs a clean entry next session**. The two Type 5 candidates this session (drained-by-socializing, isolation-as-coping) had quality-control issues. Worth a deeper Phase B pass on Type 5 angles like "intellectual arrogance / always thinks they know better" or "why do some people need so much time to recharge."
- **Stress-response is a goldmine for Enneagram** — every type has a stress arrow and a security arrow. Worth searching for more "X under stress" questions: how X behaves when overwhelmed / under pressure / triggered.
- **Avoid two-type-overlap traps** — Type 4 vs Type 7 "want more / never satisfied" already in the 04-28 queue, so today's "starting-projects" was framed pure Type 7 to avoid muddying it.
