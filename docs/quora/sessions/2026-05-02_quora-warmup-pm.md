**Score:** 78
**Decision:** Queue
**Notes:** **Closes long-standing Type 5 gap** (detached-observer / autonomy variants returned zero in last 3 sessions). Sean Simmons' top answer correctly names "shame from disparity" but never names the type-specific mechanism (Type 5 economics vs Type 8 vulnerability vs Type 1 moral).
