## Single Tweets (Gut Punches)

### Love Language Reveals

```
Your love language isn't how you want to receive love.

It's how you desperately try to earn it.

Type 2s give Acts of Service because that's how they bought approval as kids.

Type 3s need Words of Affirmation because silence felt like failure.

Same language. Different wound.
```

```
Type 4s don't need Quality Time because they're romantic.

They need it because being alone with someone who truly sees them is the only proof they're not fundamentally broken.

That's not a love language. That's a survival strategy.
```

```
"My love language is Acts of Service"

Translation:
- Type 1: "I show love by fixing your flaws"
- Type 2: "I show love so you can't leave"
- Type 8: "I show love because words make me feel weak"

Same language. Three different wounds running the show.
```

### Dating Red Flags by Type

```
Dating red flags by type:

Type 2: Helps you before you ask, then keeps score
Type 3: More interested in how you look together than who you are
Type 4: Romanticizes the chase, bored when they catch you
Type 6: Tests your loyalty like it's a full-time job
Type 8: Picks fights to see if you'll stay

Your red flag is your wound on display.
```

```
Type 9s don't ghost you because they lost interest.

They ghost you because deciding to end it feels like violence.

So they just... disappear. And convince themselves you didn't notice.

You noticed.

That's not peace-keeping. That's conflict-avoiding to the point of cruelty.
```

```
Type 3s will date who they SHOULD love, not who they DO love.

The partner that looks good on paper.
The one their parents would approve of.
The one that advances the narrative.

Then wonder why intimacy feels like a performance.

You're not in love. You're casting.
```

### Relationship Sabotage

```
How each type ruins their own relationships:

Type 1: Criticizes their partner "for their own good"
Type 2: Gives until they resent, then blames the receiver
Type 3: Prioritizes image over intimacy
Type 4: Rejects available love, chases unavailable
Type 5: Withdraws when things get emotionally real
Type 6: Tests loyalty until the other person fails
Type 7: Gets bored, finds someone "more exciting"
Type 8: Picks fights to test strength
Type 9: Merges until they disappear, then resents it

The pattern isn't bad luck. It's you.
```

```
Type 2 + Type 8 relationships:

2: "I give everything and they don't appreciate it"
8: "They try to control me with guilt"

Both are right.
Both are doing their trauma response.

The 2 gives to earn love.
The 8 rejects to avoid being owned.

Looks like chemistry. Actually trauma bonding.
```

