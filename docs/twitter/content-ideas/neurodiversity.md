### ADHD by Type

```
ADHD hits different based on your type:

Type 1 + ADHD: "Why can't I just DO the thing I know is right?"
Type 3 + ADHD: "I have 47 productivity systems and use none"
Type 4 + ADHD: "My brain is too unique for neurotypical structures"
Type 5 + ADHD: "I hyperfocused for 12 hours on the wrong thing"
Type 6 + ADHD: "What if the medication has side effects I can't predict?"

Same diagnosis. Different flavor of chaos.
```

```
Type 7 + ADHD is a cheat code for avoiding pain.

47 browser tabs.
3 unfinished projects.
Constantly planning the next thing instead of finishing this thing.

You call it "ADHD."
Your therapist calls it "elaborate avoidance."

Both can be true. The question is which one you're using as an excuse.
```

```
Type 1 + ADHD is the cruelest combination.

Internal standards: Perfectionism
Brain capacity: Chaos goblin

Every dropped ball feels like moral failure.
Every unfinished task is proof you're fundamentally flawed.

It's not executive dysfunction. It's executive dysfunction + a courtroom in your head that never adjourns.

The verdict is always guilty.
```

```
ADHD + Type 5:

Hyperfocuses for 14 hours on a niche interest.
Forgets to eat.
Forgets to respond to humans.
Becomes world expert on something irrelevant.
Still can't do the taxes.

"But I learned so much!"

You hoarded information instead of functioning. That's not productivity. That's avoidance with extra steps.
```

### Anxiety by Type

```
Type 6 anxiety vs regular anxiety:

Regular: "What if something goes wrong?"
Type 6: "I've already imagined 47 ways it could go wrong and prepared for 43 of them"

You're not anxious. You're running a full-time contingency planning department in your brain.

The preparation doesn't reduce anxiety. It feeds it.
```

```
Type 1 anxiety sounds like: "I'm doing everything wrong"
Type 2 anxiety sounds like: "Everyone needs something from me"
Type 3 anxiety sounds like: "I'm falling behind everyone"
Type 4 anxiety sounds like: "Nobody will ever understand me"
Type 5 anxiety sounds like: "I don't have enough energy for this"
Type 6 anxiety sounds like: "Everything is about to go wrong"

Same nervous system. Different soundtrack.
```

```
Type 3 anxiety:

Panic attack in the bathroom.
Fixed makeup.
Back to the meeting.
Nobody noticed.

That's not resilience. That's performance under duress.

The achievement keeps going even when you're breaking inside.

High-functioning doesn't mean okay. It means good at hiding.
```

### Autism + Masking

```
Autism masking by type:

Type 1: Over-prepares for every social situation, scripts conversations
Type 2: Focuses entirely on others to avoid being observed
Type 3: Performs neurotypical success until they can't anymore
Type 4: Reframes autism as "artistic temperament"
Type 5: Withdraws to avoid masking demands entirely

The mask isn't protection. It's a prison with a smile painted on.
```

```
Type 9 + autism:

"I'm fine with whatever"
(Translation: I can't process my own preferences over the noise)

"I don't mind"
(Translation: Making a choice takes energy I don't have)

"Whatever you want"
(Translation: I've already lost myself in trying to accommodate everyone)

The agreeableness isn't peace. It's camouflage. And it's exhausting.
```
