## Part 1: Single Tweet Best Practices

### Character Count Guidelines

- **Standard limit**: 280 characters
- **Optimal engagement length**: 71-100 characters
- **Emojis count as 2 characters** (Unicode encoding)
- **Premium users**: Up to 25,000 characters for long-form posts

### Single Tweet Structure

```
[HOOK - Attention-grabbing opener]
[VALUE - Core message or insight]
[CTA - What you want readers to do] (optional)
```

### When to Use a Single Tweet

‚úÖ **Use single tweets for:**

- Quick insights or observations
- Standalone quotes or wisdom
- Simple questions to spark discussion
- Timely reactions to trending topics
- Punchy one-liners or hot takes
- Content easily discovered via search/hashtags

‚ùå **Avoid single tweets for:**

- Complex explanations requiring multiple steps
- Stories with narrative arc
- Detailed how-to guides
- Content that benefits from visual progression

## Part 2: Thread Best Practices

### Thread Structure

**Tweet 1 (The Hook):**

```
[BOLD STATEMENT or QUESTION]
[PROMISE of what's coming]
[Thread emoji üßµ or "Thread:" indicator]
```

**Body Tweets (2-7):**

```
[NUMBER or TRANSITION] (optional)
[KEY POINT]
[SUPPORTING DETAIL or EXAMPLE]
```

**Final Tweet:**

```
[SUMMARY/TL;DR]
[CALL TO ACTION]
[LINK if applicable - put links here, NOT in first tweet]
```

### Optimal Thread Length

- **Sweet spot**: 4-8 tweets
- **Maximum recommended**: 10-12 tweets before attention drops
- **Minimum**: 2-3 tweets (otherwise use single tweet)

### When to Use a Thread

‚úÖ **Use threads for:**

- Step-by-step guides and tutorials
- Storytelling with beginning, middle, end
- List-based content (tips, lessons, resources)
- Breaking down complex topics
- Announcing new content with key highlights
- Building authority on a subject
- Trending topic deep-dives

‚ùå **Avoid threads for:**

- Simple announcements
- Questions seeking quick input
- Content that doesn't build on itself
- Topics that can be expressed in one tweet

### Thread Formatting Tips

1. **Number your tweets** (1/, 2/, 3/ or 1. 2. 3.) for easy navigation
2. **Use line breaks** liberally - walls of text kill engagement
3. **Keep each tweet self-contained** enough to be understood if quoted
4. **Add visuals** to 2-3 tweets within the thread
5. **End each tweet at a natural curiosity point** to encourage continued reading

## Part 3: The Hook (Critical for Both)

The hook is the **make-or-break element**. Top creators write 10-15 versions before publishing.

### 6 Hook Types That Work

#### 1. Strong Declarative Statement

```
"Twitter is the most underrated business tool in the world."
"I spent 10 years building the wrong things."
```

#### 2. Thought-Provoking Question

```
"What if everything you know about productivity is wrong?"
"Why do 90% of startups fail in year one?"
```

#### 3. Controversial Opinion

```
"College completely failed in teaching me how to write."
"Most business advice is designed to keep you busy, not successful."
```

#### 4. Moment in Time

```
"In 2019, I was $50,000 in debt with no plan."
"Yesterday I got rejected for the 100th time."
```

#### 5. Vulnerable Statement

```
"I've never admitted this publicly, but..."
"The thing that terrifies me most about building in public:"
```

#### 6. Weird/Unique Insight

```
"The best writers I know don't actually enjoy writing."
"Here's the personality trait most billionaires share (it's not what you think):"
```

### The 4-Part Hook Formula

1. **Bold Statement** - Surprise or shock them
2. **Tension** - Highlight a struggle they relate to
3. **Twist** - Offer unexpected insight
4. **Credibility** (optional) - Show proof/receipts

**Example:**

```
"I grew from 0 to 100K followers in 8 months.

But here's what nobody tells you about 'going viral':

It almost destroyed my mental health.

Here's the real playbook I wish I had (üßµ)"
```

## Part 4: Call-to-Action (CTA) Techniques

### CTA Effectiveness Data

| CTA Type        | Engagement Increase |
| --------------- | ------------------- |
| Ask for Reply   | +334%               |
| Ask for Retweet | +311%               |
| Ask for Follow  | +258%               |

### Best CTA Practices

1. **Spell out "RETWEET"** - performs 2x better than "RT"
2. **Put CTA at the beginning** when possible (not always at end)
3. **Use CAPS** for action words to make them stand out
4. **Leverage reciprocity**: "I gave you X for free, all I ask is..."
5. **Create urgency**: "Don't miss out" beats "Get access"
6. **Use social proof**: "Join 10,000+ others who..."

### CTA Templates

**For Engagement:**

```
"If this resonated, RETWEET the first tweet to help others find it."
"Reply with your biggest takeaway üëá"
"Which tip will you try first? Comment below."
```

**For Growth:**

```
"FOLLOW for daily [topic] insights."
"If you want more [benefit], hit that follow button."
"I share [X] every week. Follow to never miss one."
```

**For Traffic:**

```
"Full guide in the link below üëá"
"Want the complete playbook? Link in reply."
"Get the free template here: [link in final tweet]"
```

## Part 5: Algorithm Optimization

### What the Algorithm Rewards

| Signal                       | Weight/Impact                                                      |
| ---------------------------- | ------------------------------------------------------------------ |
| Replies                      | **Highest** - 150x boost when you reply to comments on your tweets |
| Retweets                     | 2x more valuable than likes                                        |
| Time-on-post                 | Longer = more reach                                                |
| Visual content               | Videos get 10x engagement                                          |
| Premium/Verified             | 2-4x visibility boost                                              |
| Posting when audience active | Higher ranking                                                     |

### Visibility Boosters

1. **Reply to comments within first hour** - massive boost
2. **Engage before posting** to "wake up" your network
3. **Use native media** (upload directly vs. links)
4. **Avoid external links in first tweet** - put in reply or follow-up
5. **Use quote tweets** instead of plain retweets for more reach
6. **Post threads** - algorithm loves time-on-content

### What Hurts Visibility

- External links in the first tweet
- Too many hashtags (stick to 1-2)
- Posting when audience is inactive
- Low engagement in first hour
- Walls of text without line breaks

## Part 6: Hashtag Strategy

### Best Practices

- **Use 1-2 hashtags maximum** per post
- **Optimal hashtag length**: ~6 characters (single word)
- **Place at end** of tweet or integrate naturally
- **Choose moderately popular** hashtags (not oversaturated)
- **Threads don't need hashtags** in every tweet - just the first

### When to Skip Hashtags

- The recommendation algorithm can surface content without hashtags
- Overly promotional content looks spammy with hashtags
- Well-established accounts get reach from followers, not discovery

- **B2B content**: Weekdays during business hours (especially 1-3 PM lunch)
- **Younger audiences**: More active in evenings

## Part 10: Common Mistakes to Avoid

### Content Mistakes

‚ùå Cramming too much into each tweet
‚ùå No clear hook or value proposition
‚ùå Walls of text without line breaks
‚ùå Starting threads without a strong first tweet
‚ùå Putting links in the first tweet (kills reach)
‚ùå Using too many hashtags (3+)
‚ùå Being too promotional without providing value first

### Engagement Mistakes

‚ùå Not replying to comments (especially in first hour)
‚ùå Posting at random times
‚ùå Ignoring analytics and what works
‚ùå Not engaging with others before posting
‚ùå Using only text when visuals would help

### Thread-Specific Mistakes

‚ùå Making threads too long (10+ tweets without strong narrative)
‚ùå Not numbering tweets
‚ùå Each tweet doesn't stand alone enough
‚ùå No clear ending or CTA
‚ùå Burying the best content too deep in thread

## Part 12: AI Agent Instructions

When helping craft tweets/threads:

### Information to Gather

1. **Topic/subject** - What is the tweet about?
2. **Goal** - Engagement, traffic, followers, or conversation?
3. **Format preference** - Single tweet or thread?
4. **Tone** - Professional, casual, provocative, educational?
5. **Target audience** - Who should engage?
6. **CTA desired** - What action do you want readers to take?
7. **Any links/visuals** to include?

### Output Format

Provide the user with:

1. **Multiple hook options** (3-5 variations)
2. **Complete tweet/thread** ready to copy-paste
3. **Character count** for each tweet
4. **Suggested posting time** based on their timezone
5. **Optional variations** (different angles/tones)

### Quality Checks

Before presenting final output:

- Verify character counts are within limits
- Ensure hook is attention-grabbing
- Check for clear value proposition
- Confirm natural reading flow
- Validate CTA is appropriate for goal
- Remove unnecessary hashtags
- Suggest visual content if applicable

### Crafting Custom Prompts

**Step 1: Identify the core emotion/concept**

- What's the psychological truth you're visualizing?
- What would make someone stop scrolling?

**Step 2: Choose the visual metaphor**

- How can a Greek statue embody this concept?
- What action, pose, or setting communicates it?

**Step 3: Add emotional depth**

- What facial expression captures this?
- What's the internal experience?

**Step 4: Set the mood with color/lighting**

- What accent color fits the emotion?
- Dramatic? Moody? Soft? Ethereal?

**Step 5: Apply the technical template**

- Add the standard Unreal Engine/Cinematic parameters
- Use --ar 16:9 for Twitter
- Use --stylize 200 for consistent quality
