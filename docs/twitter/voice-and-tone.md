## The Vibe Check

Before posting, ask:

1. **Would a Type 1 block you?** (If no, add more edge)
2. **Would a Type 4 propose marriage?** (If no, add more depth)
3. **Would someone screenshot this for their group chat?** (If no, make it more specific)
4. **Would this work with zero context?** (If no, cut the setup)
5. **Is there a "but here's why" moment?** (If no, add the hug)
