# 9takes Worldbuilding

Brand worldbuilding program based on Caleb Ralston's "7 Elements of Worldbuilding for Personal Brands": Named Concepts, Re-defined Terms, Weaknesses, Characters & Settings, Interests & Values, Lore & Legends, and Common Enemy.

| File                                                                                 | What it is                                                                                    |
| ------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------- |
| [9TAKES_WORLDBUILDING_MAP.md](9TAKES_WORLDBUILDING_MAP.md)                           | Current 9takes world map and decision log                                                     |
| [WORLDBUILDING_TEMPLATE.md](WORLDBUILDING_TEMPLATE.md)                               | Reusable workflow copied into this repo and localized for 9takes                              |
| [reference-buildos/](reference-buildos/)                                             | Full BuildOS reference set copied into this repo; use as the quality bar, not as 9takes canon |
| [ARCHETYPE_CONFLICT_POSTER.md](ARCHETYPE_CONFLICT_POSTER.md)                         | Persona map for people bringing a real situation to 9takes                                    |
| [ARCHETYPE_PERSONALITY_ANALYSIS_READER.md](ARCHETYPE_PERSONALITY_ANALYSIS_READER.md) | Persona map for visitors entering through famous-person analysis                              |
| [ARCHETYPE_SOCIAL_NAVIGATOR.md](ARCHETYPE_SOCIAL_NAVIGATOR.md)                       | Persona map for people using 9takes tactically in dating, friendship, work, and conflict      |
| [content-queue/](content-queue/)                                                     | Post/blog/social ideas generated by the worldbuilding run                                     |
| [enneagram-8/](enneagram-8/)                                                         | One-off Type 8 worldview/worldbuilding run                                                    |

Start with [9TAKES_WORLDBUILDING_MAP.md](9TAKES_WORLDBUILDING_MAP.md). The next practical step is to ratify the naming hierarchy in its decision log, then wire the chosen names into the homepage, question flow, personality-analysis CTAs, and social content system.
