# Anger, fear, shame: the three alarms in every fight

**Status:** queued
**Lane:** Education / social
**Source:** `docs/brand/9takes-manifesto.md`, `docs/positioning/enneagram-how-to-think-about-it.md`.

## Hook

Most fights are three alarms arguing over which one is allowed to be real.

## Core idea

The Enneagram gets accessible when it starts with anger, fear, and shame instead of wings, instinctual variants, and type debate.

## Product tie-in

Foundation post for social carousels, question prompts, and onboarding.

## Draft angle

Use quick real-world examples: Type 8 directness, Type 6 reassurance-seeking, Type 3 image pressure, Type 1 correctness.
