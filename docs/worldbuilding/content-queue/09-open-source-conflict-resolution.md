# Open source conflict resolution

**Status:** queued
**Lane:** Questions platform
**Source:** `docs/brand/brand-positioning.md`, [ARCHETYPE_CONFLICT_POSTER.md](../ARCHETYPE_CONFLICT_POSTER.md).

## Hook

One friend's advice is one lens. Nine takes gives you the spread.

## Core idea

Conflict becomes easier to navigate when the emotional reads are visible side by side. 9takes turns private confusion into a structured perspective set without making one person the judge.

## Product tie-in

Use for `/questions`, onboarding, and community-page copy.

## Draft angle

Compare "asking the group chat" against asking a room designed to preserve different lenses.

## Dependency

Best after the question flow can show fuller Nines.
