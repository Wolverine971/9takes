# Rest You Don't Have To Earn

**Status:** queued
**Surface:** blog / social
**Source:** [ENNEAGRAM_1_WORLDVIEW_BIBLE.md](../ENNEAGRAM_1_WORLDVIEW_BIBLE.md)

## Hook

The body made it to the beach. The court stayed in session.

## Core idea

For Ones, rest functions as a wage paid out after perfection, and perfection never quite arrives, so rest keeps its receipt. The redefinition: rest is maintenance, not a reward. Part of the machine working, not payment for the machine finishing.

## Structure

1. The unstartable vacation.
2. The wage model of rest and why the paycheck never clears (the moving goalpost).
3. Rest as maintenance: what changes when it stops being earned.
4. One experiment: rest before the list is done, once, and log that nothing degraded.

## Product tie-in

Question page: "What's the rest you keep postponing until everything is right?" SP1-flavored but universal reach.
