# Stay For The Second Layer

**Status:** queued
**Surface:** carousel / coaching prompt
**Source:** [ENNEAGRAM_7_WORLDVIEW_BIBLE.md](../ENNEAGRAM_7_WORLDVIEW_BIBLE.md)

## Hook

The moment something stops being exciting is not always the moment it stops being right.

## Core Idea

The Unfinished Middle begins after the spark fades and before repetition reveals craft, intimacy, consequence, or hidden novelty. The Type 7 experiment is to stay one beat past the automatic exit and inspect the second layer.

## Structure

1. First-layer novelty.
2. The dip that gets called boredom.
3. What only the middle can reveal.
4. Define a reversible stay period.
5. Reassess with new information, not reflexive relief.

## Product Tie-in

Question prompt: "What becomes interesting if you stay for the second layer?"

## Guardrail

Staying is an experiment, not a command to tolerate harm, exploitation, or a clearly wrong fit.
