# The Siege

**Status:** queued
**Surface:** stress/growth explainer
**Source:** `src/blog/enneagram/enneagram-type-8.md`

## Hook

When a Type 8 breaks, the fighter goes quiet.

## Core idea

Type 8 stress often looks like a move to Five: withdrawal, secrecy, information hoarding, cold analysis, and preparing for betrayal. This is not the absence of anger. It is anger locked inside a fortress.

## Structure

1. The usual Eight is visible.
2. The stressed Eight disappears.
3. The trigger is usually powerlessness, betrayal, dependence, or humiliation.
4. The repair question: what exact freedom, trust, or line felt threatened?
5. Growth: return to direct contact before the siege becomes identity.
