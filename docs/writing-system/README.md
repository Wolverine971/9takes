## Workflow Overview

```
Phase 1: Strategy & Planning
    └── Fill out planning template

Phase 2: Structure First (McPhee Method)
    └── Choose structural pattern

Phase 3: Drafting for Substance
    └── Write following 9takes formula

Phase 4: Two-Pass Editing
    ├── Pass 1: Macro (structure)
    └── Pass 2: Micro (language)

Phase 5: SEO & Technical
    └── Optimize for search

Phase 6: Final Review & Publish
    └── Quality checks and launch
```

---

## The 9takes Formula (Quick Reference)

1. **Emotional Hook** - Start with feeling
2. **Pattern Recognition** - "You've probably noticed..."
3. **Psychological Insight** - "Here's why..."
4. **Type-Specific Application** - "For Type X..."
5. **Practical Tools** - Scripts, frameworks, exercises
6. **Clear Next Step** - Specific action

---

## Quality Standards

### Must-Haves

- Clear value proposition
- Actionable takeaways
- Type-specific insights
- Problem-focused title
- Quick Answer box
- 3-5 internal links

### Never Include

- Hedging language ("tend to", "might")
- Academic/theoretical tone
- Walls of text
- Unsubstantiated claims
- Generic descriptions
