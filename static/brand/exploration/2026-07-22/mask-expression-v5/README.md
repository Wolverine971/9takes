## Generation method

Images were generated with the built-in image-generation tool in identity-preserving edit mode. Every expression prompt repeated the same invariants: exact master identity, outer silhouette, front view, proportions, material, background, lighting, camera, crop, and scale. Only brow, eyelid, cheek, mouth, jaw, nostril, and very small attitude cues were allowed to change.

The shared creative direction was:

> Passion in the expression; virtue in the restraint.

Expression intensity was held around 30–40%. Prompts explicitly prohibited horror, caricature, culturally specific makeup, teeth, text, decoration, and everyday clichés such as money for avarice, food for gluttony, seduction for lust, or sleeping for sloth.

## Review note

This is a concept-validation set, not final production artwork. Type 2 versus Type 3 was the closest collision pair in the first assembly. The selected Pride replacement now uses visible cheek lift and shallow crescent eye apertures, while Deceit retains neutral eye and cheek planes with a mouth-only smile. The contrast is clear at review size and meaningfully improved at icon size. Minor material/silhouette drift should be normalized only after the expression choices are approved.

No live application assets were changed.
