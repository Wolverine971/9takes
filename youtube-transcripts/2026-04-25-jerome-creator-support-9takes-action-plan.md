# 9takes Action Plan — Applying the Jerome / Creator Support Lessons

> **Companion document.** The full transcript breakdown and 17 tips are in [`2026-04-25-jerome-creator-support-1200-videos-ANALYSIS.md`](./2026-04-25-jerome-creator-support-1200-videos-ANALYSIS.md). This doc translates that into concrete moves for **9takes**.

---

## Diagnosis: where 9takes is "Jerome"

Jerome's pattern: **1,200 videos, 150–200K views, no household-name brand.** Volume is up, identity is fuzzy, no one can describe him in a sentence to a friend.

9takes' parallel symptoms:

| Jerome's Symptom | 9takes Equivalent |
| --- | --- |
| Stacks 4 storylines in the first minute | Personality-analysis blogs open with type definition, behavior list, *and* a celebrity profile, all at once |
| "Lessons are commodity, your relationship to lessons is the moat" | Encyclopedic Enneagram explainers compete with Truity, Enneagram Institute, Reddit. Where's DJ's POV? |
| Buries personal story at the end out of fear | DJ's origin (why 9takes exists, how Enneagram changed his life) is buried — not load-bearing in the funnel |
| Filters audience too early ("scratch golfer / zero handicap") | Type-numbered headlines ("Type 5 in Relationships") filter out the ~95% of people who don't know their type yet |
| No tagline that travels word-of-mouth | "See the emotions behind every take" is a *line*, not yet a *meme*. No reader-friendly sign-off baked into every piece |
| Treats data as strategy | Volume of celebrity blogs is up; brand legibility hasn't compounded with it |
| Treats every celebrity blog as a standalone | No recurring "cast," no callbacks across pieces, no shared universe |

The reframe: **9takes doesn't need more output. It needs identity that compounds.**

This is the line we want fans repeating verbatim. No variation, no synonyms.

## The 9takes Identity × Emotion × Action worksheet (Tip #4)

Apply by **content vertical**, not by individual post. Each row is a contract you make with the reader.

| Vertical | Audience identity | Emotion to feel | Action to take |
| --- | --- | --- | --- |
| `/personality-analysis` (celebrity) | Pop-culture obsessive who half-knows their type | "I just *got* why she does that" | Read the related-celebrity → take the test → submit a question |
| `/enneagram-corner` | Self-development reader, suspicious of woo | "This is sharper than the other Enneagram stuff I've read" | Subscribe → read another type breakdown |
| `/community` (The Takes of 9takes) | Existing 9takes user looking for emotional validation | "Someone finally said it" | Leave a comment / contribute a take |
| `/pop-culture` | Casual scroller via Google search | "Wait, is X a Type 8?" | Click into related-personality → submit guess |
| `/how-to-guides` | High-intent searcher with a problem | "This is concrete enough to actually try" | Bookmark → return → eventually book a session |
| `/book-session` | Type-aware reader at a turning point | "I want this dialed in" | Join waitlist |
| `/questions` | Anonymity-seeker with a private dilemma | "I can be honest here" | Give first → read others |

**Action:** Pin this table to `docs/brand/`. When writing/editing a piece, check: *Does this title, opener, and closing serve this row?* If not, cut, rewrite, or move the piece.

## Concrete moves, mapped tip-by-tip

### Move 1 — Restructure the celebrity-blog opener (Tips #1, #2, #9)

**Today:** Most personality-analysis blogs open with type explainer + bullet list of behaviors + biographical context, all in the first 200 words.

**Fix:** Lead every celebrity blog with **one hook** — the universal version a non-Enneagram-reader gets instantly.

Template:

```
H1: <Celebrity Name>'s Enneagram Type, Decoded

Lede (3 sentences max):
- One specific behavior or moment that puzzles fans.
- The Enneagram answer in plain English (no type number yet).
- "Once you see it, you can't unsee it." — promise of the read.

Then drip:
  → Now name the type.
  → Now show the supporting evidence.
  → Now zoom out to "what this means for you if you're Type X."
  → Close with a recurring sign-off (see Move 7).
```

**Specific cut:** strip "Type X is the Y" from the first sentence of every published celebrity blog. Move it to paragraph 2 or 3. This removes the early filter Colin warned Jerome about.

**First targets to apply:** the 5 celebrity drafts currently dirty in git (Cardi B, Emma Myers, Matt Rife, Sandra Bullock, Shakira). Use them as the template before pushing.

### Move 2 — Build the "decentralized sitcom" cast (Tip #6)

**Idea:** Treat the celebrity roster as a connected universe. Recurring characters, callbacks, in-jokes.

**Specific deliverables:**

1. **"The Type 8 Squad" / "Type 5 Lab" / etc. landing pages** — one per type, listing every famous person 9takes has typed, with cross-links. Each page is a hub readers can browse the way Marvel fans browse character lists.
2. **"Crossover" posts** — *"What Happens When a Type 8 (Cardi B) and a Type 4 (Lana Del Rey) Get in a Room?"* These are pure entertainment + maximally shareable + cost almost nothing to produce because the type analyses already exist.
3. **Persistent in-blog callouts** — every celebrity post includes a `<RelatedTypes>` callout: "Other Type Xs we've decoded: A, B, C." Already partially done via `RelatedPosts` component (`src/lib/components/blog/`), but needs **type-clustered** logic, not just chronological "more from this section."
4. **Internal "ranking" / "tier list" content** — *"The 9 Most Misread Type 8s in Hollywood"* uses existing assets, plays to the universe, and is highly word-of-mouth-friendly.

**Action items:**
- [ ] Build `/personality-analysis/types/[1-9]` hub pages (driven by `blogs_famous_people` DB queries grouped by type).
- [ ] Add `relatedSameType` field to the personality-analysis `+page.server.ts` load.
- [ ] Plan one "crossover" or "tier list" piece per month for Q3 2026.

4. Gets riffed into a Quora answer, an Instagram carousel, and a homepage hero subhead.

**Why this works:** Tip #13 — vulnerability isn't manufactured but it is structural. Right now DJ's story isn't structurally placed in the funnel. It should appear on first contact.

### Move 4 — A 9takes "tagline closer" (Tip #12)

Colin's example: end every video with *"Whatever you're pursuing in your life, impossible is possible."*

**9takes equivalents to test (DJ picks one):**

- *"One situation, 9 ways to see it."*
- *"There are 9 takes. Yours is one of them."*
- *"See the emotions behind every take."* (current tagline, but make it a *closer*, not just a hero line)
- *"What's your take?"* (interrogative, invites reply — dangerous-good for word-of-mouth)

**Where it goes:**
- Last line of every blog, above the comment widget.
- Last line of every Instagram caption.
- Last line of every Quora answer (where allowed).
- Last line of welcome email and weekly digest.

**Rule:** It is the *same line*, every time. No variation. Repetition is the feature, not a bug.

### Move 5 — The "knock them off the list" series (Tip #7)

Jerome's coach list = a **named, finite, public** quest viewers can root for.

**9takes versions:**

1. **"100 People, 9 Types"** — a public, ongoing project to type the 100 most-searched/most-influential people of 2026. List published, names crossed off as we publish, public progress bar on the page. Every individual celebrity blog becomes part of a *bigger story*.
2. **"The Hard-to-Type List"** — a public list of celebrities who are notoriously contested (Taylor Swift, Beyoncé, Elon Musk). DJ commits to a take on each, with the reasoning. The contested-ness is the engine.
3. **"DJ's Cold-DM Quest"** — a periodic post where DJ asks a famous person on a podcast if their type matches our analysis. Even unanswered, the *attempt* is the content. Story-within-the-story (Tip #7's "do you know this guy?" mechanic).

**Action items:**
- [ ] Publish the "100 People, 9 Types" landing page with the full target list visible on day 1.
- [ ] Add a progress visual (X/100) to the homepage and to each celebrity blog footer.

### Move 6 — Define what 9takes' "POV moat" is (Tip #5)

This is the highest-leverage move and the hardest. Right now the personality-analysis content is *good* but largely interchangeable with what a careful Truity reader could write.

**The 9takes-specific moat to lean into:**

- **Pattern-recognition over taxonomy.** Don't just say "Cardi B is a Type 8." Say *"watch how she pivots from threat to humor in a single sentence — that's the Type 8 self-protective reflex collapsing into the playful counterphobic move. Here's where you'll see it in your own life."*
- **Emotional logic, not personality boxes.** The brand promise is "see the emotions behind every take" — every blog should pay that off in a way no other Enneagram site does.
- **DJ's read on contradiction.** When a celebrity does something off-type, *that's the gold.* "She's a clean Type 4, except when X. Here's what that exception tells us about Type 4s under pressure."

**Action item:** Add a required section to every personality-analysis post template: **"Where she breaks the type — and what that reveals."** This is a section no Truity-clone has, and it's where DJ's POV lives.

---

### Move 7 — Stop burying the personal beats (Tip #13)

In every long-form piece, DJ should be **named, present, and stake-bearing in the first ~150 words.** Not just "according to the Enneagram." Instead: *"I've been wrong about Type 4s twice in my life. Both times it cost me a relationship. Here's what I've learned reading them in public figures since."*

**Concrete change to the blog content templates:** Add a "DJ's stake" callout block (similar to `QuickAnswer.svelte` in `src/lib/components/blog/callouts/`) that lives in paragraph 2 of every long-form piece. Mandatory, not optional.

---

### Move 8 — The "stakes" rewrite for Q&A landing pages (Tip #9)

The `/questions` flow has the give-first mechanic, which is powerful but invisible to first-time visitors. Before they see the question wall, they see a UI obstacle.

**Fix the framing:** Before the give-first prompt, seed *why* it exists. One paragraph in DJ's voice:

> *"On most Q&A sites, you can lurk forever. Here, you have to give an answer first. I built it that way because I noticed that the answers I valued most in my life came from people who'd been in the same room I was in. Lurkers don't know what it costs. Contributors do. Give first — then you'll see what 9 different types said."*

This is Tip #9: stake-seeding before the action. Without it, give-first feels like a paywall. With it, it feels like a value system.

## What success looks like (Colin's bar, applied to 9takes)

Per Tip #14: even if 9takes does nothing strategic, traffic will grow with reps. The harder, more valuable bar is the **qualitative** one Colin pushed Jerome on:

- Can a stranger describe 9takes in one sentence? *(Right now: probably not.)*
- Does anyone repeat the tagline back to DJ unprompted? *(Right now: no.)*
- Is there a single celebrity blog that reads unmistakably like 9takes — one no other Enneagram site could have written? *(That's the moat post. Build to it.)*

The aim is not 2× traffic. It's becoming the site people **name when asked.** Volume is already on track. Identity is the lever.
